PestControl Admin Panel Site Plan

Site Name Purpose Scenarios Colors Typography Wireframes

Project Overview

The project is a simple web application prototype for a pest control business. The goal is to help a business owner organize clients, services, technicians, and service records in a cleaner way than using Excel spreadsheets.

The final website will contain three main pages:

  • index.html: Login page for owner/admin and technician access.
  • dashboard.html: Admin page with dynamic forms for clients, services, and technicians.
  • records.html: Organized lists of clients, services, employees, and technician service updates.

Site Name

PestControl Admin Panel

This name was selected because it clearly describes the purpose of the website. It is an administration panel designed specifically for a pest control business. The name is simple, professional, and easy for a business owner or employee to understand.

Optional domain idea: pestcontroladmin.com

Site Purpose

The purpose of the site is to provide a simple digital management system for a pest control company. The owner or admin will be able to create and manage clients, services, and technicians. The system will also allow technicians to view assigned services, add service notes, and mark services as completed.

The website will demonstrate dynamic JavaScript functionality by displaying forms, generating lists from data, using local storage, opening modal dialogs, and updating content based on user interaction.

Scenarios

These are possible questions from the target users of the website:

  • How can I quickly create a new client and assign a pest control service to that client?
  • How can a technician see which services are assigned to them and mark them as completed?
  • How can the owner review all clients, employees, and services in one organized place?
  • How can the business keep better service records instead of using Excel?

Color Scheme

The selected color scheme uses green as the main color because it connects with pest control, safety, nature, and cleanliness. Orange is used as an accent color to highlight important actions.

Primary Green

Hex: #14532d

Used for headings, header background, navigation text, buttons, and important sections.

Accent Orange

Hex: #f97316

Used for call-to-action buttons, highlights, borders, and active states.

Light Background

Hex: #f8fafc

Used for the page background to keep the interface clean and readable.

Typography

The main font selected for this project is Inter, with Arial and Helvetica as fallback fonts. Inter was selected because it is clean, modern, readable, and commonly used in professional dashboard interfaces.

Heading Example

Headings will use a strong font weight to clearly separate sections and improve readability.

Body text will use a regular font weight with enough line spacing to make forms, instructions, service details, and records easy to read.
